# 自定义登录页面样式指南

## 🎨 如何修改登录页面的颜色和名称

### 第一步：修改软件名称和标语

打开文件：`src/frontend/src/pages/login/login.vue`

找到第 19 行，修改软件名称：

```vue
<!-- 原始代码 -->
<h1 class="logo-text">OmniAgent</h1>

<!-- 修改为你的软件名 -->
<h1 class="logo-text">智能助手Pro</h1>
```

找到第 21 行，修改标语：

```vue
<!-- 原始代码 -->
<p class="subtitle">开启您的智能协作新纪元</p>

<!-- 修改为你的标语 -->
<p class="subtitle">让AI成为你的得力助手</p>
```

### 第二步：修改颜色主题

在同一个文件中，找到 `<style>` 部分（第 164 行开始）

#### **方案1：蓝色主题**

```scss
/* 1. 修改背景颜色 */
.login-container {
  background-color: #e3f2fd;  /* 浅蓝色背景 */
}

/* 2. 修改背景装饰气泡 */
.blob-1 {
  background: #bbdefb;  /* 浅蓝色 */
}

.blob-2 {
  background: #90caf9;  /* 蓝色 */
}

.blob-3 {
  background: #e3f2fd;  /* 极浅蓝色 */
}

/* 3. 修改卡片边框和阴影 */
.login-card {
  border: 1px solid rgba(100, 181, 246, 0.5);  /* 蓝色边框 */
  box-shadow: 0 20px 40px rgba(33, 150, 243, 0.1);  /* 蓝色阴影 */
}

/* 4. 修改输入框样式 */
.custom-input {
  :deep(.el-input__wrapper) {
    border: 1px solid #bbdefb;  /* 蓝色边框 */
    
    &:hover {
      border-color: #42a5f5;  /* 悬停时深蓝色 */
    }
    
    &.is-focus {
      border-color: #42a5f5;  /* 聚焦时深蓝色 */
      box-shadow: 0 0 0 4px rgba(33, 150, 243, 0.1) !important;
    }
  }
  
  :deep(.el-input__prefix-inner) {
    color: #42a5f5;  /* 图标蓝色 */
  }
}

/* 5. 修改登录按钮 */
.login-btn {
  background: linear-gradient(135deg, #42a5f5 0%, #1e88e5 100%);  /* 蓝色渐变 */
  
  &:hover {
    box-shadow: 0 10px 20px rgba(33, 150, 243, 0.3);  /* 蓝色阴影 */
  }
}

/* 6. 修改链接颜色 */
.forgot-pwd {
  color: #42a5f5;  /* 蓝色 */
  
  &:hover {
    color: #1e88e5;  /* 深蓝色 */
  }
}

.register-footer {
  :deep(.el-button--link) {
    color: #42a5f5;  /* 蓝色 */
    
    &:hover {
      color: #1e88e5;  /* 深蓝色 */
    }
  }
}

/* 7. 修改版本标签 */
.version {
  color: #1976d2;  /* 深蓝色文字 */
  background: #e3f2fd;  /* 浅蓝色背景 */
}
```

### 第三步：修改 Logo 图片

1. 准备你的 Logo 图片（建议使用 SVG 格式，支持缩放不失真）
2. 将图片放到 `src/frontend/src/assets/` 目录
3. 修改第 17 行的代码：

```vue
<!-- 原始代码 -->
<img src="../../assets/agentchat.svg" alt="OmniAgent" class="logo-img" />

<!-- 修改为你的 Logo -->
<img src="../../assets/my-logo.svg" alt="我的软件" class="logo-img" />
```

## 📝 修改步骤总结

1. **修改软件名称**：找到 `<h1 class="logo-text">` 标签
2. **修改标语**：找到 `<p class="subtitle">` 标签
3. **修改颜色主题**：在 `<style>` 部分替换颜色代码
4. **修改 Logo**：替换 `<img src="...">` 的路径
5. **保存文件**：Ctrl+S（Windows）或 Cmd+S（Mac）
6. **刷新浏览器**：查看效果

## 💡 提示

- 所有颜色都使用十六进制格式（如 `#42a5f5`）
- 渐变色使用 `linear-gradient()` 函数
- 阴影使用 `box-shadow` 属性
- 边框使用 `border` 属性
- 建议使用 Material Design 配色方案，视觉效果更专业
